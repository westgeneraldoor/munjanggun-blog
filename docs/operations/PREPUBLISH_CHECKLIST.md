# 발행 전 체크리스트

> 목적: 네이버 블로그에 붙여넣기 전에 발행 사고를 막는다.  
> 적용 대상: 신규 글, 리라이팅 발행본, 중요 보강 글

---

## 1. 파일 구조

- [ ] `posts/` 파일은 발행 본문만 담고 있다.
- [ ] 첫 줄이 `# 제목 후보`가 아니다.
- [ ] `검색 의도 분석`, `썸네일 카피`, `품질 채점 결과`, `예상 성능`이 본문 파일에 남아 있지 않다.
- [ ] 제작 노트는 `outputs/drafts/NNN_키워드_note.md`에 보존돼 있다.
- [ ] 자동 검수 결과는 `outputs/checks/NNN_키워드_check.md`에 저장돼 있다.

## 2. 제목

- [ ] 최종 제목 1개가 명확하다.
- [ ] 제목에 숫자가 포함돼 있다.
- [ ] 질문형 제목이 아니다.
- [ ] 타겟 키워드가 앞쪽에 있다.
- [ ] `중문` 단독 제목이 아니다.
- [ ] 최근 5편과 같은 제목 패턴이 과도하게 반복되지 않는다.

## 3. 본문 품질

- [ ] 검색자의 실제 고민이 도입부에 드러난다.
- [ ] 현장 기준, 사례, 비교표 중 최소 1개 이상이 있다.
- [ ] 문장군 제품명과 시공 가능 범위가 정확하다.
- [ ] 근거 없는 고정 가격이나 과장 수치를 쓰지 않았다.
- [ ] `[사진:` 플레이스홀더가 발행 본문에 남아 있지 않다.

## 4. 금지 표현

- [ ] `물론`, `또한`, `더불어`, `이처럼`, `결론적으로`를 쓰지 않았다.
- [ ] `안녕하세요 문장군입니다`로 시작하지 않는다.
- [ ] `보양`, `현관문`, `무조건`, `완벽 방음`, `완전 방음`이 없다.
- [ ] 불가 지역을 가능 지역처럼 쓰지 않았다.

## 5. 링크와 CTA

- [ ] 실제 네이버 내부링크가 최소 2개 이상 있다.
- [ ] `현재 발행 예정`, `기존 000번`, `등록된 000번` 같은 임시 문구가 없다.
- [ ] 무료 방문실측 또는 네이버 예약 CTA가 자연스럽게 들어가 있다.
- [ ] 광고처럼 강한 권유가 아니라 편한 예약 안내 톤이다.

## 6. 해시태그

- [ ] 해시태그는 5~10개다.
- [ ] 해시태그에 띄어쓰기가 없다.
- [ ] `#문장군`, `#문장군중문`이 포함돼 있다.
- [ ] 본문 중간이 아니라 태그란 입력용 섹션으로 분리돼 있다.

## 7. 명령어 검수

발행 전 아래 명령을 실행한다.

```bash
npm run validate:posts
```

중요 글이나 리라이팅 발행본은 아래 명령까지 확인한다.

```bash
npm run validate:posts:strict
```

경고가 남아도 발행할 수는 있지만, 경고 사유를 알고 발행해야 한다.
