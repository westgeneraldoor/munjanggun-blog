# 문장군 일일 SEO 관제 루틴

> 목적: 매일 들어오는 네이버 블로그 성과 데이터를 가볍게 기록하고, 다음 발행 순서와 신규 글감 판단에 반영한다.

## 운영 원칙

- 사장님 입력 부담은 5분 이내로 유지한다.
- 채팅으로 받은 데이터는 `outputs/reports/daily/YYYY-MM-DD_seo_watch.md`에 남긴다.
- 원본 엑셀이나 관리자 화면 이미지는 공개 저장소에 커밋하지 않는다. 로컬 보관은 가능하지만 `data/naver/daily/`, `data/naver/raw/`, `data/naver/**/*.xlsx`는 Git 추적 금지다.
- 저장소에는 익명화한 요약 리포트(`outputs/reports/daily/YYYY-MM-DD_seo_watch.md`)만 남긴다.
- 하루 데이터로 결론을 확정하지 않고, 7일 누적과 14일 누적을 우선 판단 기준으로 본다.
- 현관문/방화문처럼 제외 결정된 소재는 유입이 있어도 글감으로 확장하지 않는다.
- 신규 글감은 일일 유입어만으로 확정하지 않는다. `docs/operations/TOPIC_SELECTION_SCORECARD.md`에 따라 네이버 광고 API 시장 수요와 블로그 실제 반응을 함께 본다.
- daily report는 증거 로그다. 보고서의 다음 액션은 `docs/strategy/ACTIVE_TOPIC_QUEUE.md`에서 보호, 발행대기, scorecard 필요, 관찰, 제외 중 하나로 닫는다.

## 매일 받으면 좋은 정보

1. 전날 전체 조회수
2. 전날 공감수, 댓글수, 이웃증감수
3. 유입경로 비율
4. 상세 검색어 목록
5. 게시글 조회수 TOP 20
6. 그날 발행한 글 URL
7. 필요 시 특이사항

중요: 게시글의 실제 작성일/발행일은 네이버 통계의 `게시글 조회수 TOP 20` 표에 보이는 `작성일`을 기준으로 한다. 채팅으로 URL을 받은 날짜는 URL 등록일일 뿐이다.

## 매일 데일리 현황 보고 범위

기존 원고 소진 기간에는 아래 항목을 매일 하나의 현황 보고로 묶는다.

| 구분 | 확인 내용 | 처리 |
| --- | --- | --- |
| 업로드 링크 | 당일 발행한 포스팅 3개 URL | `POSTING_REGISTRY.md` URL 등록 |
| 기본 통계 | 조회수, 공감, 댓글, 이웃증감 | 일일 관제 기록에 입력 |
| 유입 분석 | 유입경로, 상세 검색어, 게시글 TOP 20 | 키워드 클러스터 분류 및 `작성일` 확인 |
| 총괄 판단 | 보호 글, 보강 글, 신규 후보, 제외 후보 | 다음 발행/신규 기획에 반영 |

보고 제목은 아래 형식을 사용한다.

```text
YYYY-MM-DD 문장군 데일리 현황 보고
```

## 기존 원고 소진 기간 운영

현재 URL 대기 원고를 빠르게 소진하는 기간에는 신규 원고 작성을 서두르지 않는다.

운영 순서:

1. 하루 3개씩 기존 원고를 발행한다.
2. 발행 URL을 받으면 `POSTING_REGISTRY.md`에 등록한다. 단, 발행일은 URL 수신일이 아니라 TOP 20 표의 `작성일`로 맞춘다.
3. 같은 날 또는 다음날 통계 자료를 받아 일일 관제 기록을 작성한다.
4. 기존 원고 소진 후, 누적 데이터 기반으로 신규 포스팅 주제를 다시 선정한다.

핵심 원칙:

- 기존 원고 소진 중에는 새 글감을 무리하게 늘리지 않는다.
- 단기 반응은 참고만 하고, 신규 글감 결정은 누적 데이터로 한다.
- 발행 속도는 높이되, URL 등록과 통계 기록은 빠뜨리지 않는다.
- URL을 먼저 받았더라도 이후 TOP 20 `작성일`과 다르면 `POSTING_REGISTRY.md`의 발행일을 TOP 20 기준으로 보정한다.

## 순위 체크 운영 기준

순위 체크는 데일리 필수 항목에서 제외한다.

이유:

- 네이버 블로그탭 순위만으로 통합검색 실제 유입을 대표하기 어렵다.
- 개인화, 기기, 시점에 따라 흔들림이 있다.
- 현재 데일리 통계의 유입경로/상세 검색어/게시글 TOP 20이 실제 운영 판단에 더 직접적이다.
- 50개 키워드 전체 순위 체크는 시간이 오래 걸려 매일 운영 비용 대비 효율이 낮다.

운영:

| 주기 | 범위 | 목적 |
| --- | --- | --- |
| 매일 | 순위 체크 생략 | 실제 유입과 게시글 반응 중심 판단 |
| 주 1회 | 핵심 키워드 10~15개 | 보호 글 하락, 신규 글 반영 여부 확인 |
| 월 1~2회 | 전체 키워드 | 리라이팅 후보와 전략 포트폴리오 재조정 |

## 기록 파일 규칙

```text
outputs/reports/daily/2026-06-15_seo_watch.md
outputs/reports/topic_candidates/2026-06-15_topic_scorecard.md
data/naver/daily/2026-06-15/  # 로컬 전용, Git 커밋 금지
```

일일 리포트에는 아래 4개만 남긴다.

| 구분 | 내용 |
| --- | --- |
| 핵심 지표 | 조회수, 공감, 댓글, 순위/채널 진단 |
| 검색어 원본 | 주요 검색어와 비율 |
| 총괄 해석 | 강한 키워드군, 새로 뜬 키워드, 제외 키워드 |
| 다음 액션 | 발행 우선순위, 리라이팅 후보, 신규 글감 후보 |

## 키워드 점수 알고리즘

키워드는 아래 방식으로 우선순위를 판단한다.

```text
키워드 점수 =
광고 API 시장수요
+ 유입비율
+ 반복등장 점수
+ 구매의도 점수
+ 기존글성과 점수
+ 글감공백 점수
- 제외키워드 패널티
- 중복소재 패널티
```

### 점수 항목

| 항목 | 기준 |
| --- | --- |
| 광고 API 시장수요 | 월간 검색량, 모바일 비중, 경쟁 강도, 허브/클러스터/롱테일 확장성이 좋으면 가산 |
| 유입비율 | 상세 유입경로에서 비율이 높을수록 가산 |
| 반복등장 | 여러 게시글/여러 날짜에 반복되면 가산 |
| 구매의도 | 가격, 비용, 견적, 설치, 교체, 업체, 시공 포함 시 가산 |
| 기존글성과 | 이미 TOP 20 안에 들어온 글과 연결되면 가산 |
| 글감공백 | 기존 글에서 아직 독립적으로 다루지 않은 각도면 가산 |
| 제외키워드 | 현관문, 방화문, 미제공 제품, 불가 지역이면 감점 또는 제외 |
| 중복소재 | 이미 같은 비교/체크리스트를 다뤘으면 감점 |

상세 점수표와 후보 보고 형식은 `docs/operations/TOPIC_SELECTION_SCORECARD.md`를 따른다.

## 운영 명령어

매일 통계 정리 후 아래 명령어를 실행한다.

```powershell
npm run ops:daily
```

`ops:daily`는 아래를 확인한다.

| 항목 | 처리 |
| --- | --- |
| daily report 존재/필수 섹션 | 누락 시 FAIL |
| topic scorecard 산출물 | 1차 운영에서는 누락 시 WARN |
| 키워드 데이터 최신성 | 누락 시 FAIL, 오래되면 WARN |
| active topic queue | 컬럼/상태/금지 키워드/발행대기 링크 누락 시 FAIL |

순위 리포트와 TOP10 분석은 daily 필수에서 제외하고 주간 보조 점검으로 분리한다.

```powershell
npm run ops:weekly
```

## 판단 등급

| 등급 | 의미 | 처리 |
| --- | --- | --- |
| A | 바로 매출형 글감 | 신규 글 또는 발행 순서 상향 |
| B | 기존 글 보강/내부링크용 | 리라이팅, 내부링크, 소제목 보강 |
| C | 관찰 필요 | 7일 반복 여부 확인 |
| D | 제외/보류 | 확장하지 않음 |

## 매일 총괄 판단 순서

1. 유입경로에서 네이버 통합검색 비중을 확인한다.
2. 게시글 TOP 20을 비용형, 비교형, 마감재형, 방문/도어형으로 분류한다.
3. 상세 검색어를 A/B/C/D로 나눈다.
4. 발행 대기 글 중 당겨야 할 글을 찾는다.
5. 신규 글감 후보를 3개 이하로만 기록한다.
6. 다음날 사장님에게 필요한 추가 정보가 있으면 1~2개만 요청한다.

## 2주 후 자동화 후보

2주간 같은 양식으로 리포트가 쌓이면 아래 작업을 검토한다.

- 네이버 유입 엑셀 자동 파싱
- 검색어 클러스터 자동 분류
- A/B/C/D 점수 자동 계산
- `CONTENT_PLAN.md` 발행 우선순위 추천
- 주간 SEO 요약 리포트 자동 생성
