## 1. 키워드 전략 구조: 3계층 피라미드

```
         ┌─────────────────────┐
         │   1계층: 허브 글     │  ← 대형 키워드 (7,000~35,000/월)
         │   5개 종합 가이드    │     트래픽 유입의 중심축
         └──────────┬──────────┘
                    │ 내부링크
         ┌──────────▼──────────┐
         │  2계층: 클러스터 글   │  ← 중형 키워드 (1,000~5,000/월)
         │  20개 세부/전환 글    │     실질적 예약/견적 전환 발생
         └──────────┬──────────┘
                    │ 내부링크
         ┌──────────▼──────────┐
         │  3계층: 롱테일/지역   │  ← 소형 키워드 (100~1,000/월)
         │  10개 틈새 공략 글    │     경쟁 낮음 + 전환율 극대
         └─────────────────────┘
```

**원칙:** 모든 하위 글은 반드시 상위 허브 글로 내부링크 → SEO 권위 집중

- 현관 중문 종류 비교 / 우리집에 맞는 중문 찾기 / 좁은 현관 해법
