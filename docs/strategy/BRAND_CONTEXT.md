## 실측 견적 상담 (영업부 매니저)

- 레이저 레벨기로 정확한 수직·수평 사이즈 측정 + 구조 진단
- 시공이 어려운 현장도 최대한 가능한 방법을 찾아냄
- 태블릿으로 영상·디자인 소재 상담
- 유리 샘플북 + 컬러 샘플북으로 실물 확인 후 견적
- 결정 시 네이버 브랜드스토어 결제 방법·옵션 안내
- 리뷰 이벤트 참여 방법 가이드
- **공실인 경우:** 링크+사진+유선상담으로 동일 서비스 제공
- **담당제:** 실측→제작→시공→A/S까지 담당매니저 1인이 끝까지 책임

## 시공 운영

### 시공 규모 & 시간
- **일 시공:** 평균 25현장, 최대 35현장 (전속 시공팀 3팀이 분담)
- **3연동중문:** 약 1~3시간
- **ABS도어(문짝교체):** 약 30분/1짝
- **시공 가능 시간:** 오전 9시~오후 4~5시 (대부분 그 전에 완료)

### 제작~시공 기간
- **중문:** 고객 결정 후 약 3~4일 후 시공
- **도어:** 고객 결정 후 약 1주일 후 시공

### 시공 후
- 시공 후 뒷정리 진행
- ⚠️ 보양 작업은 별도로 하지 않음 (블로그에서 "보양 작업 한다"고 쓰면 안 됨)

### 시공 제약
- **토요일:** 시공은 하지만 아파트 등에서 시공 금지하는 경우 있음
- **공휴일(빨간날):** 전체 휴무
- **이른 아침/늦은 오후:** 민원 우려로 시공 불가

#### 페르소나 A — 이사 없는 리모델링파 (핵심 타겟)
- **프로필:** 30~40대, 5~15년 거주 아파트, 이사 계획 없음
- **상황:** 체리색 문짝이 낡고, 화장실 문은 습기로 부풀어 오름
- **핵심 고민:** "살면서 시공 가능한가?", "먼지/소음 얼마나?", "짐 다 빼야 하나?"
- **검색어:** 방문교체 (9,720), 화장실문교체 (4,270), 문짝교체비용 (2,310), 방문교체비용 (1,940)
- **결정 포인트:** 시공 시간 짧음, 무료 실측, 시공 후 뒷정리
- **CTA 경로:** 블로그 → 네이버 예약 실측 접수

#### 페르소나 B — 인테리어 초보 신혼부부
- **프로필:** 20~30대, 첫 자가/전세, 인테리어 관심 높음
- **상황:** 현관에 중문 달고 싶은데 종류가 너무 많아서 뭘 골라야 할지 모름
- **핵심 고민:** "어떤 중문이 우리 집에 맞지?", "가격이 얼마나 하지?", "비싸면 어쩌지?"
- **검색어:** 아파트중문 (5,280), 3연동중문 (14,450), 중문가격 (1,800), 아파트중문설치비용 (1,610)
- **결정 포인트:** 비교표, 투명한 견적 구조, 리뷰 수
- **CTA 경로:** 블로그 → 브랜드스토어 결제 또는 네이버 예약

#### 페르소나 C — 셀프 시도 후 전문가 의뢰
- **프로필:** 30~50대, DIY 시도했다가 실패한 경험
- **상황:** 셀프 교체하려다 경첩 안 맞거나 문틀 손상, 전문업체 찾는 중
- **핵심 고민:** "전문가한테 맡기면 얼마나 더 드나?", "실패한 부분 복구 가능한가?"
- **검색어:** 셀프방문교체 (310), 셀프중문 (1,300), ABS도어 (7,260)
- **결정 포인트:** 셀프 vs 전문가 비용 비교, A/S 범위
- **CTA 경로:** 블로그 → 전화상담 또는 네이버 예약

## 고객 FAQ TOP
1. 가격이 얼마예요? → **실측 후 정확한 견적, 특이사항까지 미리 안내**
2. 우리 집도 시공 가능한가요? → **서비스 지역 내 아파트 대부분 가능, 어려운 현장도 최대한 방법 찾음**
3. 먼지/분진/소음 걱정, 민원은? → **시공 후 뒷정리 + 중문 1~3시간 내 완료**
4. 추가금이 나오나요? → **실측 때 특이사항 확인 후 미리 안내, 현장 갑작스런 추가금 없음**
5. 시공 시간/기간은? → **중문 1~3시간, 방문 30분/1짝 / 중문 결정 후 3~4일, 도어 1주일 내 시공**
6. 서비스 가능 지역인가요? → **서비스 지역 명확히 안내**
7. A/S는? → **도어 3년, 중문 2년, 자체 시공팀 직접 방문 처리**

| 상세페이지를 읽어도 학습이 잘 안 됨 | 블로그는 긴 스펙 설명보다 한 현장의 문제와 판단 기준을 쉽게 설명해야 한다 |

### 무료 방문 실측을 추천해야 하는 이유

고객은 가격을 알고 싶어하지만, 실제로는 `내 집에 가능한가`, `추가금이 생기는가`, `최종 견적이 얼마인가`를 한 번에 알고 싶어한다. 그래서 문장군 블로그는 가격표를 단정하기보다 무료 방문 실측에서 무엇을 확인하는지 설명해야 한다.

무료 방문 실측에서 확인하는 것:

- 현장 구조 확인
- 레이저 레벨기로 수직·수평 확인
- 실측 사이즈 확인
- 시공 가능 여부 확인
- 추가금 여부 안내
- 마감 방식 안내
- 색상, 유리, 디자인 옵션 상담
- 최종 견적가 현장 안내
- 진행 시 결제 가이드, 시공일 예약, 공사 안내문, 리뷰 이벤트, 네이버 예약 후기 작성 가이드까지 안내

### 중문 시공 가능 여부 판단 기준

| 제품/구조 | 문장군 현장 기준 |
| --- | --- |
| 3연동중문 | 벽과 벽 사이가 1m를 넘으면 검토 가능. 최대 약 2m까지 가능. 단, 중문틀이 온전히 세워질 벽공간이 필요하다 |
| 3연동중문 초과 폭 | 2m를 넘으면 2짝미서기양개 또는 3연동양개형을 검토 |
| 원슬라이딩중문 | 최소 약 800mm도 검토 가능. 다만 문이 옆으로 제쳐질 공간과 방해물이 없는지가 중요 |
| 스윙중문 | 대략 900~1500mm 범위에서 검토 |
| 신발장 간섭 | 중문틀이 벽에 고정될 공간을 신발장이 침범하면 문제. 신발장리폼으로 해결 가능한지 확인 |
| 좁은 현관 | 안 되는 제품에 미련을 두기보다 가능한 범위 안에서 시공 가능한 구조를 먼저 봐야 한다 |

중요: 위 수치는 블로그에서 `무조건 가능`처럼 쓰지 않는다. "현장 구조에 따라 검토 가능"으로 표현한다.

### 현장에서 자주 생기는 추가 구조 변수

중문 견적이 달라지는 이유는 제품 가격만이 아니다. 아래 변수는 블로그 비용 글에서 반드시 현장 기준으로 설명한다.

- 디자인 옵션
- 유리 옵션
- 사이즈 차이
- 기존 중문 철거와 폐기물 처리
- 기둥 보강
- 바닥 보강
- 스위치 간섭과 매립형 스위치등박스
- 가벽 필요 여부
- 신발장리폼
- 천장몰딩, 걸레받이, 우물천장 등 마감 간섭

가장 자주 언급할 수 있는 현장 변수는 `기둥 + 바닥보강`이다.

### 사진 요청 기준

상담 전 사진은 사이즈보다 구조를 먼저 보기 위한 자료다.

고객에게 요청하면 좋은 사진:

- 바닥과 천장이 모두 보이게 최대한 멀리서 찍은 현관 전체 사진
- 중문이 들어갈 위치 전체가 보이는 사진
- 신발장 문을 열었을 때 간섭이 보이는 사진
- 스위치, 몰딩, 바닥 단차, 가벽, 기둥 위치가 보이는 사진

블로그에서는 `[사진:]` 지시문을 쓰지 않고, 본문 문장으로 자연스럽게 사진 방향을 유도한다.

### 최저가·저가 시공 글에서 강조할 기준

고객은 최저가를 보면 "이게 실구매가인가", "시공까지 포함인가", "추가금은 없는가"를 궁금해한다. 문장군 글은 가격만 비교하지 말고 서비스 경험까지 설명해야 한다.

좋은 표현:

- 가격은 제품만이 아니라 상담, 실측, 안내, 시공, 마감, 사후관리까지 포함해 봐야 한다.
- 좋은 시공 경험은 제품만의 문제가 아니라 진행 과정이 얼마나 편하고 문제 없이 이어지는지의 차이다.
- 문장군은 현장에서 추가금 여부와 마감 방식을 미리 안내하는 쪽으로 운영한다.

피해야 할 표현:

- 무조건 최저가
- 추가금 0원 보장
- 전화로 확정 견적 가능
- 누구나 셀프 설치 가능

### 자재 판매·셀프중문 기준

문장군은 중문 자재 판매를 진행하지 않는다. 업자가 아닌 일반 고객이 중문 자재만 사서 직접 설치하면 품질이 떨어지거나 현장 대응이 어려워질 가능성이 크다.

셀프중문 글의 방향:

- 셀프를 무조건 비난하지 않는다.
- 다만 중문은 문짝만 사는 문제가 아니라 틀, 벽공간, 수평, 바닥, 신발장, 스위치, 마감이 함께 맞아야 한다고 설명한다.
- 무료 방문 실측은 "셀프 시공을 하라고 안내하는 절차"가 아니라 "우리 집 구조와 비용을 정확히 확인하는 안전한 상담"으로 안내한다.

### 문장군이 하지 않는 것과 주의 표현

| 항목 | 기준 |
| --- | --- |
| 9mm 문선 | 문장군은 12mm 슬림문선으로 안내한다. 9mm는 비교 검색어로만 다루고, 문장군 시공 가능 서비스처럼 쓰지 않는다 |
| 9mm vs 12mm | 육안 차이는 크지 않지만, 9mm는 얇아 찍힘·눌림 위험이 있다. 9mm가 대중적인 이유는 가격과 보급성이지 품질 우위가 아니다 |
| 문틀만 단독 교체 | 기존 문틀보다 새 문틀이 작게 들어가 기존 문짝을 그대로 달기 어렵다. 문짝도 새로 해야 한다 |
| 공틀 마감 | 문짝이 필요 없고 문틀만 마감하려는 경우 쌈지 마감은 가능 |
| 비대칭양개형중문 | 문장군 미취급. 가능 제품처럼 쓰지 않는다 |
| 중문파티션 | 단독 소재로 확장하지 않는다. ㄱ자중문은 가능 |
| 현관문/방화문 | DEC-017에 따라 블로그 확장 금지 |

### ABS도어·방문교체 판단 기준

| 고객 질문 | 문장군식 답변 방향 |
| --- | --- |
| 문짝만 교체와 문짝+문틀세트 차이 | 고객이 가장 많이 헷갈리는 부분이다. 글에서는 교체 범위를 먼저 구분한다 |
| 문짝만 교체 | 대부분 가능하다. 문짝을 현장 문틀에 맞춰 조정한다. 단, 문틀이 심하게 틀어진 경우 빛샘 가능성을 사전 안내한다 |
| 문짝+문틀세트 | 비거주중, 도배·장판 공사가 함께 있으면 가능하다. 비용이 커질 수 있다 |
| 화장실문 세트 교체 | 타일공사 계획이 없으면 기존 문틀 철거 때 타일 면이 깨질 수 있다. 이 경우 PVC마감재가 현실적인 마감 방식이다 |
| 멤브레인도어 vs ABS도어 | 고객에게는 "나무문 vs 반영구 방수문"처럼 쉽게 설명한다 |

### 거주중 시공 기준

거주중 고객의 핵심 걱정은 먼지, 분진, 소음, 집 안 상처다. 글에서는 겁주는 방식이 아니라 "무엇을 치우고, 어디에서 작업하고, 얼마나 걸리는지"를 알려줘야 한다.

| 항목 | 문장군 기준 |
| --- | --- |
| 중문 시공 시간 | 3연동, 원슬라이딩, 스윙 모두 대략 2~3시간으로 넉넉히 안내 |
| 집에 있어야 하는지 | 시공보다 실측 때 있으면 좋다. 결정할 것이 많기 때문 |
| 사전 준비 | 시공 자리와 현관까지의 이동 동선, 인근 짐 정리 |
| 만족 포인트 | 처음에는 시각적 만족감, 사용하면서는 마감과 부드러운 사용감 같은 체감 만족감 |

### AppSheet 현장 사진 활용 기준

문장군은 AppSheet에 사진과 현장 정보가 라벨링되어 있어 직원이 필요한 현장을 필터로 찾을 수 있다. 글 작성자는 AppSheet 원본을 먼저 찾느라 멈추지 말고, 사진이 붙기 좋은 현장 조건을 본문에 자연스럽게 설계한다.

좋은 현장:

- 사진이 많은 현장
- 특이사항이 있는 현장
- 고객이 걱정한 걸림돌이 사진으로 보이는 현장
- Before와 After 변화가 뚜렷한 현장

블로그 독자가 관심 있는 사진:

- 시공 전 위치
- 신발장, 스위치, 기둥, 바닥 단차, 몰딩처럼 공감 가능한 걸림돌
- 기둥, 스위치등박스, 바닥보강, 천장몰딩마감, 걸레받이마감, 우물천장마감, ㄱ자중문, 신발장리폼 등 해결 결과

실측 중간 과정 자체는 독자 관심이 낮을 수 있다. 중간 과정은 "문제를 어떻게 풀었는지"가 보일 때만 쓴다.

### 문장군 신뢰를 보여주는 방식

문장군 신뢰는 "우리가 잘합니다"라고 말해서 생기지 않는다. 고객이 확인할 수 있는 지표와 운영 체계로 보여줘야 한다.

사용 가능한 신뢰 근거:

- 쇼핑 리뷰
- 네이버 예약 리뷰
- 판매수
- 리뷰 내용
- 구독·팔로우
- AppSheet 기반 운영 체계
- 카카오톡 알림톡 단계별 안내
- 영업부 견적서 알림톡
- 시공 발주, 시공 배정, 시공 완료 알림
- 담당자, 연락처, 주소, 시공 내역, 옵션, 스펙, 주의사항, 견적서 유효일자 안내

좋은 글의 목표:

> 고객이 "재밌다", "나도 무료면 견적상담 받아보고 싶다", "여기에서 하면 좋겠다"라고 느끼게 만드는 것.

겁주는 글과 현실을 알려주는 글은 다르다. 겁주는 글은 위험만 말한다. 문장군식 글은 위험인자, 해결 방법, 미리 알아채는 방법, 고객이 할 액션까지 알려주고 공감해준다.

## 세일즈 플로우
```
A) 브랜드스토어 결제 → 실측예약 → 견적 → 취소 후 재결제 → 시공
B) 전화상담 → 견적예약 → 견적 → 결제 → 시공
C) 네이버예약 → 견적예약 → 견적 → 결제 → 시공
```

## 블로그 전략
- **목표:** 상위노출 → 브랜드 인식 → 브랜드스토어 유입 / 네이버예약 접수
- **발행 빈도:** 일 1회
- **톤:** 전문가 (시공/상담/견적 전반)
- **활용 소재:** AppSheet에서 라벨링 검색한 시공 사례 사진(Before/After), 고객 리뷰
- **시즌:** 중문 = 겨울 수요↑ 여름 주춤 / 도어 = 사계절 비슷, 이사철↑

## 콘텐츠 카테고리
1. **시공 사례** — Before/After + 스펙 + 고객 반응
2. **제품 가이드** — 비교/선택 기준/가격 정보
3. **고객 Q&A** — FAQ 기반 정보성
4. **업계 노하우** — 현장 팁, 실수 방지, 체크리스트

## 말해도 되는 것 / 안 되는 것

| ✅ 말해도 되는 것 | ❌ 말하면 안 되는 것 |
