# 문장군 블로그 포스팅 제외·전환 규칙

> 목적: 통계 유입어에 제외 키워드가 떠도 새 글감 판단이 흔들리지 않도록 한다.
> 최종 업데이트: 2026-06-24

## 1. 완전 제외

아래 소재는 유입어가 있어도 신규 글감으로 확장하지 않는다.

| 소재 | 판정 | 기준 |
| --- | --- | --- |
| 현관문 | 영구 제외 | DEC-017. 사장님이 재개 지시하기 전까지 제안 금지 |
| 방화문 | 영구 제외 | 현관문 계열로 묶어 제외 |
| 비대칭양개형중문 | 미취급 제외 | 문장군 공식 취급 제품처럼 쓰지 않는다 |

## 2. 단독 글감 제외, 현장 변수로는 사용 가능

아래 소재는 단독 상품 글로 쓰지 않는다. 다만 문장군 실제 중문/도어 현장의 구조 변수나 추가 확인 항목으로는 사용할 수 있다.

| 소재 | 단독 글감 | 본문 활용 |
| --- | --- | --- |
| 신발장리폼 | 단독 글감 제외 | 중문 설치 시 신발장 돌출, 문 열림 간섭, 측면 가공, 추가금 변수로 사용 가능 |
| 가벽 | 단독 인테리어 가벽 글은 신중 | 현관 중문 설치를 위한 고정면, ㄱ자 구조, 신발장 간섭 해결 기준으로 사용 가능 |
| 걸레받이·몰딩 | 인테리어 일반론 단독 확장 주의 | 중문/도어/문선 마감 연결, 바닥·벽 경계 변수로 사용 가능 |
| 스위치매립형박스 | 단독 글감 아님 | 중문 설치 시 스위치 간섭 해결 변수로 사용 가능 |
| 바닥보강·기둥보강 | 단독 글감 아님 | 현장 특이 구조, 견적 변동 기준으로 사용 가능 |

중요: `신발장리폼`은 제외 제품이 아니다. BRAND_CONTEXT의 중문 추가금 항목에 있는 실제 현장 변수다.

## 3. 검색어는 살리되 문장군 기준으로 전환

| 유입어/소재 | 작성 전환 |
| --- | --- |
| 9mm 문선 | 비교 검색어로만 사용. 문장군 시공 가능 서비스처럼 쓰지 말고 12mm슬림문선 세트 기준으로 안내 |
| 문틀만 교체 | 문틀만 단독 교체 가능 주장 금지. 문짝+문틀세트, 문짝교체+문선마감, 문짝교체+문틀필름 기준으로 전환 |
| 접이식중문 | 소비자 표현으로만 받고 본문은 문장군 제품명인 스윙 구조로 설명 |
| 살면서리모델링 | 종합 리모델링이 아니라 살면서 방문교체, 거주중 중문시공으로 좁힘 |
| 중문파티션 | 단독 글감 제외. 고객 의도는 현관가벽, ㄱ자중문, 신발장 간섭, 고정면 기준으로 전환 |

## 4. 글감 판단 순서

1. 유입어가 완전 제외인지 확인한다.
2. 단독 글감 제외인지, 본문 현장 변수로 쓸 수 있는지 나눈다.
3. 문장군 제품명과 실제 취급 범위로 전환한다.
4. AppSheet 사진을 붙일 수 있는 현장 장면을 떠올린다.
5. 실제 고객 사례처럼 쓸 근거가 없으면 특정 고객 이야기로 꾸미지 않는다.

## 5. 작성자가 절대 헷갈리면 안 되는 것

- `중문파티션`은 제외지만, `현관가벽`과 `ㄱ자중문`은 가능하다.
- `신발장 단독 설치`는 제외지만, `신발장리폼`은 중문 현장 변수로 가능하다.
- `9mm 문선 시공`은 문장군 서비스처럼 쓰면 안 되지만, `9mm vs 12mm 비교`는 가능하다.
- `문틀만 단독 교체`는 안 되지만, `문짝+문틀세트`와 `문짝교체+문선마감`은 가능하다.
