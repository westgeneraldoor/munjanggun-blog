# 현관중문비용 갑자기 달라지는 3가지 구조 조건

현관중문비용은 중문 종류만으로 정해지지 않습니다.

같은 3연동중문을 생각해도 현관 구조, 가벽 필요 여부, 기존 중문 철거와 마감 범위에 따라 견적 기준이 달라질 수 있습니다. 그래서 비용을 비교할 때는 제품명보다 구조 조건을 먼저 봐야 합니다.

현관중문비용이 갑자기 달라지는 이유는 대부분 현장에서 확인되는 변수 때문입니다.

## 1. 현관 폭과 열림 방향이 비용 기준을 바꿉니다

현관 폭은 중문 종류를 고르는 첫 기준입니다.

하지만 폭만큼 중요한 것이 열림 방향입니다. 3연동은 레일과 프레임 위치가 필요하고, 스윙은 문이 열리는 반경을 봐야 하며, 원슬라이딩은 문짝이 이동할 벽면이 필요합니다.

| 중문 구조 | 먼저 볼 기준 |
| --- | --- |
| 3연동중문 | 레일 위치와 개방폭 |
| 스윙중문 | 문 열림 반경과 신발장 간섭 |
| 원슬라이딩중문 | 문짝이 이동할 벽면 여유 |
| ㄱ자중문 | 꺾이는 구조와 고정면 |

비용만 보고 종류를 고르면 실제 현관에 맞지 않을 수 있습니다. 구조에 맞는 방식부터 정해야 견적도 흔들리지 않습니다.

## 2. 가벽과 고정면 여부가 견적 차이를 만듭니다

현관에 중문을 고정할 면이 충분하면 구조가 단순해집니다.

반대로 오픈형 현관이거나 신발장 끝선이 애매하거나, 한쪽이 비어 있는 구조라면 가벽이나 고정면을 따로 검토해야 합니다. 이때 비용 기준이 달라질 수 있습니다.

| 구조 조건 | 비용이 달라지는 이유 |
| --- | --- |
| 고정면 충분 | 중문 프레임 설치가 비교적 단순 |
| 고정면 부족 | 가벽 또는 대체 구조 검토 |
| ㄱ자 현관 | 꺾이는 부분 마감 필요 |
| 신발장 간섭 | 리폼 또는 위치 조정 검토 |

가벽이 필요한지 아닌지는 사진만으로 단정하기 어렵습니다. 실제로 프레임을 어디에 고정할 수 있는지 확인해야 합니다.

## 3. 철거와 마감 범위를 빼면 비용 비교가 어렵습니다

현관중문비용을 비교할 때 자주 빠지는 항목이 철거와 마감입니다.

기존 중문이 있으면 철거와 폐기물 처리 여부를 봐야 합니다. 바닥 레일 자국, 천장과 벽면 마감선, 기존 몰딩 상태에 따라 마감 범위도 달라질 수 있습니다.

| 항목 | 확인할 내용 |
| --- | --- |
| 기존 중문 | 철거 필요 여부 |
| 바닥 마감 | 레일 자국과 단차 |
| 벽면 마감 | 프레임 고정과 마감선 |
| 폐기물 처리 | 철거 자재 처리 범위 |

비용이 저렴해 보여도 철거와 마감이 빠져 있으면 나중에 비교가 어려워집니다. 견적을 볼 때는 포함 범위를 같이 봐야 합니다.
