# 중문 견적 받기 전 확인할 3가지 실측 기준

중문 견적은 사진 한 장으로 끝나지 않습니다.

같은 3연동중문을 생각하고 있어도 현관 폭, 신발장 위치, 스위치 간섭, 가벽 필요 여부에 따라 견적 범위가 달라질 수 있습니다. 그래서 중문 견적을 받을 때는 제품명보다 실측 기준을 먼저 정리하는 편이 좋습니다.

중문 견적 받는 법의 핵심은 싸게 부르는 곳을 찾는 것이 아니라, 우리 집에서 어떤 변수가 견적에 들어가는지 먼저 확인하는 것입니다.

## 1. 현관 폭보다 간섭 위치를 먼저 봐야 합니다

중문은 현관 폭만 맞으면 되는 공사가 아닙니다.

문이 열리고 닫히는 방향, 신발장 손잡이, 스위치 위치, 현관 센서등, 바닥 단차까지 함께 봐야 합니다. 특히 원슬라이딩이나 스윙 구조는 문짝이 움직이는 방향이 생활 동선과 맞아야 합니다.

| 확인 항목 | 견적에 영향을 주는 이유 |
| --- | --- |
| 신발장 손잡이 | 문짝과 닿으면 구조 변경이 필요할 수 있음 |
| 스위치 위치 | 프레임 자리와 겹치면 마감 방식이 달라짐 |
| 현관 폭 | 3연동, 스윙, 원슬라이딩 선택 기준이 됨 |
| 바닥 단차 | 레일과 하부 마감 판단에 필요함 |

사진으로는 넓어 보여도 실제로 문이 움직일 자리가 부족할 수 있습니다. 그래서 견적 전에는 폭보다 간섭 위치를 먼저 보는 것이 좋습니다.

## 2. 가벽이 필요한 구조인지 확인해야 합니다

현관에 중문을 고정할 벽면이 부족하면 가벽이나 고정면이 필요할 수 있습니다.

오픈형 현관, ㄱ자 현관, 신발장 끝선이 애매한 구조에서는 중문을 어디에 고정할지부터 정해야 합니다. 이 부분이 정리되지 않으면 같은 중문이라도 견적이 달라집니다.

| 구조 | 확인 기준 |
| --- | --- |
| 오픈형 현관 | 중문을 잡아줄 고정면이 있는지 |
| ㄱ자 현관 | 꺾이는 부분을 어떻게 마감할지 |
| 신발장 옆 설치 | 가벽 없이 고정 가능한지 |
| 좁은 현관 | 가벽을 세웠을 때 더 답답해지지 않는지 |

가벽이 항상 나쁜 것은 아닙니다. 다만 꼭 필요한 가벽인지, 다른 구조로 줄일 수 있는지 실측으로 확인해야 합니다.

## 3. 유리와 프레임보다 철거와 마감 범위를 봐야 합니다

중문 견적을 받을 때 유리 종류와 프레임 색상만 먼저 고르면 중요한 항목을 놓칠 수 있습니다.

기존 중문이 있는지, 철거가 필요한지, 바닥이나 천장 마감이 어떻게 이어지는지에 따라 견적 범위가 달라집니다. 모루유리, 브론즈유리, 플루트유리처럼 유리 선택도 중요하지만 그 전에 현장 마감 조건이 먼저입니다.

| 견적 범위 | 확인할 내용 |
| --- | --- |
| 기존 중문 철거 | 철거와 폐기물 처리 여부 |
| 바닥 마감 | 레일 위치와 단차 |
| 천장·벽면 | 프레임 고정과 마감선 |
| 유리·프레임 | 디자인과 사용감 |

중문 견적은 제품 가격만의 문제가 아니라 설치 조건과 마감 범위까지 포함해서 봐야 합니다.
