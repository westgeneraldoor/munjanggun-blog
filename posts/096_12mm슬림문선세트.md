# 12mm 슬림문선 세트 선택 전 보는 3가지 기준

방문교체를 알아보다 보면 9mm 문선, 12mm 문선, 슬림문선, 무문선 같은 표현을 자주 보게 됩니다.

얇아 보이면 더 세련돼 보일 것 같지만, 문선은 두께만 보고 고르는 자재가 아닙니다. 기존 문틀 상태, 벽지 마감선, 도배 여부, 문짝과 문틀을 함께 교체하는지에 따라 선택 기준이 달라집니다.

문장군의 방문교체 기준은 12mm슬림문선 세트입니다. 9mm 문선은 비교 검색어로 설명할 수 있지만, 문장군 시공 가능 서비스처럼 안내하지 않습니다.

## 1. 얇아 보이는 것보다 마감선이 먼저입니다

문선은 문틀과 벽지가 만나는 선을 정리하는 마감재입니다.

기존 문선을 철거하면 벽지 경계, 도배선, 문틀 주변 흔적이 드러날 수 있습니다. 그래서 얇은 문선이 예뻐 보인다는 이유만으로 선택하면 벽지 마감이 어색해질 수 있습니다.

| 확인 항목 | 봐야 할 기준 |
| --- | --- |
| 기존 문선 폭 | 철거 후 흔적이 얼마나 남는지 |
| 벽지 경계 | 새 문선이 덮어줄 수 있는지 |
| 도배 여부 | 도배를 같이 하는지 아닌지 |
| 문틀 상태 | 틀어짐과 손상 여부 |

슬림문선은 얇아 보이는 느낌도 중요하지만, 기존 마감선을 얼마나 자연스럽게 정리할 수 있는지가 더 중요합니다.

## 2. 문짝만 바꿀지 세트로 볼지 나눠야 합니다

방문교체는 문짝만 바꾸는 경우와 문짝+문틀세트로 보는 경우가 다릅니다.

문짝만 교체할 때는 기존 문틀 상태를 그대로 사용합니다. 문틀이 틀어져 있거나 필름이 많이 손상되어 있으면 문짝만 새로 바꿔도 전체 느낌이 정리되지 않을 수 있습니다.

| 교체 범위 | 확인 기준 |
| --- | --- |
| 문짝만 교체 | 기존 문틀 상태가 괜찮은지 |
| 문짝+문틀세트 | 문틀 손상이나 뒤틀림이 있는지 |
| 문짝+문틀필름 | 비용과 마감 균형을 볼 때 |
| 12mm슬림문선 세트 | 문틀과 문선 마감을 함께 정리할 때 |

12mm슬림문선 세트는 문짝, 문틀, 문선 마감을 함께 보는 방식입니다. 그래서 단순히 문선 두께만 비교하기보다 교체 범위를 같이 판단해야 합니다.

## 3. 9mm와 12mm는 취향보다 시공 기준 차이입니다

9mm 문선을 찾는 이유는 대체로 더 얇고 깔끔한 인상을 원하기 때문입니다.

하지만 모든 현장에 같은 방식이 맞는 것은 아닙니다. 기존 벽면 상태, 문틀 주변 마감, 도배 여부에 따라 얇은 문선이 오히려 마감 흔적을 덜 가릴 수 있습니다.

| 비교 항목 | 확인할 내용 |
| --- | --- |
| 9mm 문선 | 얇은 인상, 별도 마감 조건 확인 필요 |
| 12mm 슬림문선 | 슬림한 느낌과 마감 덮임의 균형 |
| 무문선 느낌 | 목공과 도배 범위가 커질 수 있음 |
| 기존 문선 | 철거 흔적과 벽지 경계 확인 |

검색어는 9mm로 들어와도, 실제 판단은 우리 집 벽과 문틀 상태에 맞춰야 합니다. 문장군 기준은 12mm슬림문선 세트입니다.

## 방문교체 전에 같이 봐야 할 것

문선은 방문교체의 마지막 인상을 좌우합니다.

문짝 색상만 고르고 문선과 문틀을 놓치면 새 문을 달아도 오래된 느낌이 남을 수 있습니다. 반대로 문선까지 함께 정리하면 벽과 문 사이의 선이 더 깔끔하게 보입니다.

| 함께 볼 항목 | 이유 |
| --- | --- |
| 문짝 색상 | 방문 전체 분위기 |
| 문틀 상태 | 교체 범위 판단 |
| 문선 폭 | 벽지 경계와 마감 |
| 손잡이·경첩 | 전체 사용감 |

방문교체는 문짝 하나만의 문제가 아니라 방 전체 마감과 연결됩니다.
