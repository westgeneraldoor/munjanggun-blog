# 신축 아파트 중문 옵션 후회 줄이는 3가지 기준

신축 아파트 입주를 앞두면 중문을 옵션으로 넣을지, 입주 후 따로 설치할지 고민하게 됩니다.

옵션 중문은 입주 전에 한 번에 정리된다는 장점이 있습니다. 하지만 우리 집 현관 구조, 신발장 위치, 원하는 유리와 프레임이 맞지 않으면 입주 후 아쉬움이 남을 수 있습니다.

신축 아파트 중문 옵션은 가격만 보고 고르기보다, 나중에 바꾸기 어려운 구조부터 확인하는 게 좋습니다.

## 1. 옵션 가격보다 현관 구조가 먼저입니다

신축 아파트는 평면이 비슷해 보여도 현관 폭과 신발장 위치가 다릅니다.

중문은 문짝 하나만 놓는 공사가 아닙니다. 프레임을 고정할 벽면, 문이 열리는 방향, 신발장 손잡이와의 간섭, 스위치 위치를 함께 봐야 합니다.

| 확인 항목 | 봐야 할 기준 |
| --- | --- |
| 현관 폭 | 3연동, 스윙, 원슬라이딩 중 어떤 구조가 맞는지 |
| 신발장 위치 | 문짝 열림과 손잡이 간섭 여부 |
| 스위치 위치 | 프레임 자리와 겹치는지 |
| 벽면 유무 | 고정면이 충분한지 |

옵션 중문이든 입주 후 설치든, 결국 맞아야 하는 건 우리 집 현관 구조입니다.

## 2. 유리와 프레임 선택 폭을 봐야 합니다

신축 옵션 중문은 선택지가 정해져 있는 경우가 많습니다.

색상, 프레임 두께, 유리 종류가 마음에 들면 편합니다. 반대로 모루유리, 브론즈유리, 플루트유리처럼 원하는 분위기가 따로 있거나 집 전체 인테리어와 맞춰야 한다면 선택 폭을 비교해야 합니다.

| 선택 항목 | 확인할 내용 |
| --- | --- |
| 유리 | 투명, 모루, 브론즈, 플루트 등 |
| 프레임 | 일반, 슬림, 초슬림 느낌 |
| 색상 | 신발장, 방문, 바닥재와 어울리는지 |
| 손잡이 | 사용감과 돌출 정도 |

중문은 현관에 들어오자마자 보이는 자재입니다. 기능만큼 집 분위기와 연결되는지도 중요합니다.

## 3. 입주 후 설치가 나은 조건도 있습니다

입주 후 설치가 항상 늦거나 불리한 선택은 아닙니다.

실측을 통해 신발장 간섭, 스위치 위치, 바닥 수평, 벽면 마감을 직접 확인하고 결정할 수 있기 때문입니다. 특히 ㄱ자 구조, 좁은 현관, 가벽이 필요한 구조라면 도면보다 실제 공간 확인이 더 도움이 됩니다.

입주 후 설치를 고려할 때는 아래를 확인해보세요.

| 조건 | 확인 포인트 |
| --- | --- |
| 가벽 필요 | 고정면을 새로 만들어야 하는지 |
| 좁은 현관 | 문짝 열림과 개방폭이 충분한지 |
| 신발장 간섭 | 문이 열릴 때 닿는 부분이 있는지 |
| 디자인 맞춤 | 집 전체 색상과 유리 선택이 가능한지 |

옵션을 선택하지 않았다고 해서 중문 설치가 어려운 것은 아닙니다. 다만 구조에 맞는 방식으로 골라야 합니다.

## 신축 중문 옵션을 볼 때 피해야 할 판단

첫째, 입주 전이니까 대충 골라도 된다고 생각하면 안 됩니다.

중문은 나중에 매일 열고 닫는 자재입니다. 보기에는 예뻐도 신발장과 부딪히거나 개방폭이 좁으면 사용하면서 불편함이 생깁니다.

둘째, 옵션 가격과 개별 시공 가격만 단순 비교하면 판단이 흐려집니다. 포함된 유리, 프레임, 철거 여부, 마감 범위, A/S 기준이 다를 수 있습니다.

셋째, 유행하는 디자인만 보고 고르면 집 구조와 맞지 않을 수 있습니다. 중문은 예쁜 사진보다 실측 기준이 먼저입니다.
