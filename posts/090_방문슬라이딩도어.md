# 방문 슬라이딩도어 설치 전 보는 3가지 벽면 기준

방문을 슬라이딩도어로 바꾸고 싶어 하는 분들이 있습니다.

여닫이문이 가구에 걸리거나, 좁은 방에서 문 열리는 공간이 아깝게 느껴질 때 슬라이딩도어를 떠올리게 됩니다. 다만 방문 슬라이딩도어는 문짝만 바꾸는 문제가 아닙니다.

설치 전에는 벽면 여유, 레일 고정, 문틀과 문선 마감 기준을 먼저 봐야 합니다.

## 1. 문짝이 움직일 벽면이 있어야 합니다

슬라이딩도어는 문짝이 옆으로 이동합니다.

그래서 문 옆 벽면에 가구, 스위치, 콘센트, 몰딩이 어떻게 배치되어 있는지 확인해야 합니다. 벽면이 부족하면 문이 열릴 공간이 나오지 않거나, 가구 배치를 바꿔야 할 수 있습니다.

| 확인 항목 | 봐야 할 기준 |
| --- | --- |
| 문 옆 벽면 | 문짝이 이동할 길이가 있는지 |
| 가구 배치 | 옷장, 책상, 침대와 간섭 없는지 |
| 스위치 위치 | 레일과 문짝에 걸리지 않는지 |
| 몰딩 라인 | 문선과 걸레받이가 어색하지 않은지 |

방문 슬라이딩도어는 공간을 아껴주는 구조지만, 벽면을 사용할 수 있을 때 장점이 살아납니다.

## 2. 상부 레일과 문틀 상태를 같이 봐야 합니다

방문 슬라이딩도어는 상부 레일 고정이 중요합니다.

문짝이 움직이는 동안 흔들림이 적어야 하고, 열고 닫을 때 벽면과 바닥에 무리가 가지 않아야 합니다. 기존 문틀이 어떤 상태인지, 문선 마감이 어떻게 이어지는지도 함께 봐야 합니다.

| 기준 | 확인할 내용 |
| --- | --- |
| 상부 고정 | 레일을 안정적으로 잡을 수 있는지 |
| 기존 문틀 | 틀어짐이나 손상 여부 |
| 문선 | 슬림하게 마감할 수 있는지 |
| 바닥 | 하부 가이드 위치와 생활 동선 |

기존 방문이 오래되어 문틀이 틀어진 경우에는 단순히 문짝만 바꾸는 방식이 맞지 않을 수 있습니다. 문짝, 문틀, 문선 마감 범위를 같이 나눠야 합니다.

## 3. 방음과 사생활 기준을 기대보다 현실적으로 봐야 합니다

슬라이딩도어는 여닫이문과 닫힘 방식이 다릅니다.

문짝이 벽면을 따라 움직이는 구조라 틈새 기준이 달라집니다. 그래서 방음이나 밀폐감을 크게 기대한다면 도어 종류, 하부 가이드, 문틀 마감 방식을 함께 확인해야 합니다.

| 목적 | 확인할 방향 |
| --- | --- |
| 공간 확보 | 문 열림 반경을 줄이는지 |
| 사생활 | 유리, 패널, 문짝 재질 선택 |
| 소음 완화 | 틈새와 닫힘 방식 확인 |
| 디자인 | 문선, 벽면, 걸레받이 연결감 |

방문 슬라이딩도어는 공간 활용에 유리한 선택지입니다. 다만 방음과 밀폐감은 구조에 따라 차이가 있으니 기대치를 먼저 맞추는 게 좋습니다.

## 방문 슬라이딩도어를 피해야 할 조건

문 옆 벽면에 큰 옷장이 있거나, 침대가 바로 붙어 있거나, 스위치와 콘센트 위치가 겹치면 슬라이딩도어가 불편할 수 있습니다.
