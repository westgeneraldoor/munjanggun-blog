# 거주중 중문시공 전 꼭 보는 3가지 생활 기준

거주중 중문시공을 고민할 때 가장 먼저 떠오르는 걱정은 생활 불편입니다.

집에 있어야 하는지, 짐을 얼마나 치워야 하는지, 소음이나 분진은 어느 정도인지, 아이나 반려동물이 있으면 어떻게 해야 하는지 궁금해집니다.

거주중 중문시공은 제품만 고르는 일이 아닙니다. 시공 당일 생활 동선과 현장 준비 범위를 먼저 확인해야 불편을 줄일 수 있습니다.

## 1. 시공 공간 주변을 먼저 비워야 합니다

중문은 현관에서 작업합니다.

그래서 신발장 앞, 현관 바닥, 복도 쪽 동선이 정리되어 있어야 합니다. 현관에 유모차, 택배 박스, 신발장 앞 물건이 많으면 작업 공간이 좁아지고 이동이 불편해질 수 있습니다.

| 정리 위치 | 확인할 내용 |
| --- | --- |
| 현관 바닥 | 작업자가 움직일 공간 |
| 신발장 앞 | 문짝과 프레임 이동 동선 |
| 복도 쪽 | 자재 반입 동선 |
| 문 주변 | 스위치와 몰딩 확인 공간 |

집 전체를 비울 필요는 없습니다. 다만 중문이 설치될 현관 주변은 작업이 가능하도록 정리해두는 것이 좋습니다.

## 2. 집에 있어야 하는 시간을 확인해야 합니다

거주중 시공은 생활 중에 진행되기 때문에 일정과 시간을 미리 맞추는 것이 중요합니다.

중문은 실측 후 제작되는 제품이라 시공 당일에는 현장에서 바로 구조를 바꾸기 어렵습니다. 그래서 시공 전 실측과 상담에서 구조, 방향, 유리, 프레임 기준을 정리해두는 것이 좋습니다.

| 확인 항목 | 이유 |
| --- | --- |
| 시공 가능 시간 | 아파트 민원과 관리규정 확인 |
| 자재 반입 | 엘리베이터와 복도 동선 확인 |
| 가족 동선 | 아이, 반려동물 이동 관리 |
| 외출 계획 | 현관 사용 제한 시간 확인 |

시공 중에는 현관 출입이 불편할 수 있습니다. 그래서 택배, 외출, 아이 등하원, 반려동물 산책 시간은 미리 조정하는 편이 좋습니다.

## 3. 소음보다 현장 변수를 먼저 봐야 합니다

거주중 중문시공에서 소음과 분진 걱정은 자연스럽습니다.

다만 실제로 견적과 시공 난이도를 나누는 것은 현관 구조입니다. 신발장 간섭, 스위치 위치, 바닥 수평, 기존 중문 철거 여부가 더 큰 변수가 될 수 있습니다.

| 현장 변수 | 확인 기준 |
| --- | --- |
| 기존 중문 | 철거와 폐기물 처리 여부 |
| 신발장 간섭 | 문짝 열림과 손잡이 위치 |
| 바닥 수평 | 레일과 프레임 안정성 |
| 가벽 필요 | 고정면 확보 여부 |

거주중 시공은 빠르게 끝나는지만 볼 일이 아닙니다. 생활 중에 하는 만큼, 현장 변수를 먼저 확인해야 당일 당황할 일이 줄어듭니다.

## 거주중 시공 전에 피해야 할 생각

첫째, 사진만 보내면 견적이 정확히 나올 거라고 생각하면 어렵습니다. 사진은 구조를 보는 데 도움은 되지만, 수직과 수평, 간섭 위치는 실측이 필요합니다.

둘째, 짐을 하나도 안 치워도 된다고 생각하면 작업 동선이 부족할 수 있습니다. 현관 주변만이라도 정리하는 것이 좋습니다.

셋째, 소음만 걱정하고 구조를 놓치면 견적 판단이 흐려질 수 있습니다. 중문은 현관 구조와 마감 범위를 같이 봐야 합니다.
